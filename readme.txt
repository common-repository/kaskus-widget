Add KASKUS Widget to your websites. Official KASKUS plugin.

== Description ==

Add KASKUS widget to your WordPress site by using the plugin. Associate your WordPress site with a free KASKUS widget which display the latest hot threads and your Profile Thread from KASKUS.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->Widget Name screen to set the plugin
4. Choose the height and type that you want to display it in your website.

== Frequently Asked Questions ==

= How do I sign up for a KASKUS Account to display the widget? =

You may not sign up or sign in to use this widget.

=How to set the height above 1000 and below 250? =

You may not set the height above 1000 and below 250. the range of height is between 250 to 1000.

=How to set the my Thread to show in the widget? =

You may fill the form by this order:
	1. Widget Type (ex: Profile)
	2. User Id (ex: 3)
	3. List Type (ex: Thread)
	4. Height (ex: 320)
	5. Display Type (Image)
